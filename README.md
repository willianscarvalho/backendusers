# API backend Users
============================
## Sobre
Projeto API de Usuarios por niveis de acesso 

## Instalando

Com o node instalado rodar o npm i, para baixar os modulos por npm

Para instalá-lo em sua máquina faça os comandos a seguir:

``` bash
  npm install
  npm start
```
